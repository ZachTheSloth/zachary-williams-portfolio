GRID-BASED OS

JANUARY 2016

I created this design for fun, to explore what a grid-based operating system might look like. Each application would be made to fit within a grid, to avoid overlapping windows. This would be ideal for touch-based interfaces, such as tablets or other devices with large screens.

One potential issue with a grid-based operating system like this would be the inability to display many applications at once. It sacrifices this to prevent the need for dragging around windows, which is cumbersome in a touch-based interface. However, for simpler devices it would retain the ability to multitask with multiple applications open.
